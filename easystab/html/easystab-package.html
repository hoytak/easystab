easystab-packageR Documentation

Clustering stability analysis using Bayesian perturbations.

Description

A collection of functions for analyzing the stability of one or more clusterings. The technique used is given in [[PAPER]]. Functions are given to: assess the behavior of a clustering under perturbations, estimate the correct number of clusters in a dataset, assess the relative stability of individual clusters within a dataset, and interface with the hclust and kmeans clustering packages.

Details

The exact method used is to perturb the cluster-to-point distances by scaling them with a shifted exponential random variable, then computing the probabilities of membership for each of the points under this perturbation. This is compared against a set of randomly sampled bootstrapped baselines to determine the final stability score.
